Easily filter which content shows on Facebook Instant Articles.

== Description ==

This plugin allows you to easily select which content is shown on Facebook Instant Articles. 

You have the ability to allow articles under the following conditions:

* Show only articles that have been selected by checking a checkbox on the content editing page (All content types!)
* Show articles only from certain content types (post, page, or even custom content types)

These options disable the checkbox and custom content type options above

* Show articles only containing certain tags
* Show articles only containing certain categories
* Deny articles only containing certain tags
* Deny articles only containing certain categories

Support available via: http://consultingwp.com/?so=iaf_support

== Installation ==

= From your WordPress dashboard =
* Visit 'Plugins > Add New'
* Search for 'Instant Articles Filter'
* Activate the plugin on your Plugins page

= From WordPress.org =
* Download Instant Articles Filter
* Upload the uncompressed directory to '/wp-content/plugins/'
* Activate the plugin on your Plugins page

= Once Activated =
* Click on the 'Instant Articles Filter' menu and configure your filters
